Maze Game - Who's a Good Puppy?!

We created and built a maze game from the ground up. Selecting the theme, wireframing the design, creating svg graphics and using jquery for the dom manipulation and event handling to speed up the project build.

Cat Selector App - Top Cat

Using elements of a public service API (The Cat Api) and PHP we created an application which allows the user to select a cat breed and display all relevant images for that breed.

Aptitude Testing Application

Using a pre-built RESTful API, we built a Es6 application to test people's aptitude for Mayden Academy. We compiled to Es5 using Babel through Gulp and made use of Fetch to interact with the back-end API.

MySQL Collectors App - The Gardeners Collection

Using a combination of front-end html, css and back-end PHP, along with the design and build of a mysql database, I created an application that displays a garden collection of plants.

Background

I am an enthusiastic and driven professional with a passion for personal development and ongoing learning. I have spent the last decade working in marketing for a range of businesses, and recently reskilled as a full stack software developer.

Many of my roles and projects have been related to technology, specifically with B2B software development companies and service providers. I have always enjoyed being involved in this space and have loved getting an understanding of the domain as part of my core role and being actively involved in technology projects.

I recently graduated from Mayden Academy a full stack, intensive 16 week course, where I learnt the fundamentals of programming, became a certified scrum master and worked in a team of 8 to produce a number of projects.
